## Preventing AI Spaghetti

The number one failure mode is architectural drift across sessions. Fix with: max file lengths, explicit error handling patterns, dependency review rules, and test-alongside requirements in CLAUDE.md. Skills provide reusable procedure templates for consistent patterns.

<AnalogyBox emoji="📖" title="SOPs for AI">
Skills are Standard Operating Procedures for Claude Code. Whether creating an API endpoint or database migration, the skill ensures consistent patterns every time.
</AnalogyBox>
