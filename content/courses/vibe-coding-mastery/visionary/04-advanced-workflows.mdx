## Multi-Agent, MCP, and CI/CD

Claude Code is a general-purpose terminal agent: git workflows, shell commands, API orchestration, release management. MCP servers extend reach to databases, GitHub, and cloud services. Skills give you transparent, auditable instructions; MCP gives live external access.
