### Week 9 Build: On-Device Image Classifier

**Deliverables:**
1. Pre-trained model (MobileNet/EfficientNet) converted to Core ML .mlmodel format
2. Simple iOS app running the model locally
3. Benchmark report: inference time, accuracy, model size vs Cloud Vision
4. Decision document: should SnapSense adopt on-device classification?

> **Interview soundbite**: *"I benchmarked cloud vs on-device inference for a production mobile app — measuring latency, accuracy, cost, and privacy trade-offs. The data-driven architecture recommendation demonstrates I make technical decisions with business impact analysis, not just technical preference."*
