## Build & Exercises

### Week 7 Build: AI Risk Classification Tool

Build a web app that classifies AI use cases by EU AI Act risk tier.

**Deliverables:**
1. Working Next.js app with Claude API classification
2. Input: AI use case description → Output: risk tier + required controls + mapped NIST functions
3. SnapSense classification documented as a test case
4. At least 10 test cases covering all 4 tiers

> **Career positioning**: *"EU AI Act enforcement starts August 2026. I built a classification tool that maps AI use cases to EU risk tiers, required controls, and NIST AI RMF functions. Companies are scrambling for this capability right now."*
