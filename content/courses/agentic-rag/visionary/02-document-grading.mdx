## LLM-as-Judge

The core innovation: use an LLM to evaluate whether each retrieved document actually answers the question. Documents that fail grading get filtered out before generation.

<AnalogyBox emoji="🔬" title="Academic Peer Review">
A researcher doesn't cite every paper they find — they evaluate each source for relevance. Agentic RAG does the same with retrieved documents.
</AnalogyBox>
