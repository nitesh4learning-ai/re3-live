## Week 2 Build: Full Agentic RAG Pipeline

Extend your Week 1 Research Agent into a complete Agentic RAG system with document grading, hallucination checking, and ChromaDB storage.

### Architecture

<CodeBlock language="text" label="Agentic RAG Pipeline" code={`┌─────────┐
│  START  │
└────┬────┘
     │
┌────▼──────┐
│   Route   │ ← Decide: vector store or web search
└────┬──────┘
     │
┌────▼──────┐
│ Retrieve  │◄──────────────────┐
└────┬──────┘                   │
     │                          │
┌────▼──────┐            ┌──────┴──────┐
│   Grade   │───────────►│  Rewrite    │
└────┬──────┘ <2 relevant └─────────────┘
     │ ≥2 relevant
┌────▼──────┐

     │                          │

     │ score ≥ 0.7         score < 0.7
┌────▼────┐
│   END   │
└─────────┘`}/>

### Requirements Checklist

1. Load documents into ChromaDB with sentence-transformer embeddings
2. Router node decides between vector retrieval and web search
3. Grade each retrieved document with LLM-as-judge (binary: relevant/irrelevant)
4. If fewer than 2 relevant docs, rewrite query and retry (max 3 retries)
5. Generate answer from relevant docs only
6. Hallucination check scores grounding (0-1 scale)
7. If grounding score < 0.7, regenerate with stricter prompt
8. Track sources for citation

### Data Loading Script

<CodeBlock language="python" label="Seed ChromaDB with Sample Documents" code={`import chromadb
from chromadb.utils import embedding_functions

client = chromadb.PersistentClient(path="./chroma_db")
embedding_fn = embedding_functions.SentenceTransformerEmbeddingFunction(
    model_name="all-MiniLM-L6-v2"
)

collection = client.get_or_create_collection(
    name="knowledge_base",
    embedding_function=embedding_fn
)

# Seed with sample documents about AI governance
documents = [
    "The NIST AI Risk Management Framework provides four core functions: Govern, Map, Measure, and Manage for responsible AI deployment.",
    "ISO 42001 establishes requirements for AI management systems using Plan-Do-Check-Act methodology aligned with ISO 27001.",
    "The EU AI Act classifies AI systems into four risk tiers: Unacceptable, High, Limited, and Minimal risk.",
    "Agentic AI systems require special governance considerations including tool-use auditing and multi-step action approval.",
    "Master Data Management ensures golden record quality through matching, merging, and survivorship rules.",
]

metadatas = [
    {"source": "nist.gov", "topic": "governance"},
    {"source": "iso.org", "topic": "governance"},
    {"source": "eu-ai-act.com", "topic": "regulation"},
    {"source": "anthropic.com", "topic": "agents"},
    {"source": "gartner.com", "topic": "mdm"},
]

collection.add(
    documents=documents,
    metadatas=metadatas,
    ids=[f"doc_{i}" for i in range(len(documents))]
)

print(f"Loaded {collection.count()} documents into ChromaDB")`}/>

### Stretch Goals

1. **Multi-source routing**: Route to different ChromaDB collections based on query topic (governance → governance_collection, technical → technical_collection)
2. **Batch document grading**: Send all docs in one LLM call instead of grading individually — reduces cost by ~80%
3. **Citation tracking**: Include source URLs in the final answer, traced back to specific documents
4. **LangSmith observability**: Add LangSmith tracing to see the full execution flow and grade distribution
