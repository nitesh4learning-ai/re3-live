## Connecting Agents to the Real World
Once you build an MCP server, any MCP-compatible agent can use it — Claude Desktop, LangGraph agents, VS Code Copilot. The server code stays the same. Configure servers in `claude_desktop_config.json`, and agents discover tools automatically.
<AnalogyBox emoji="🌐" title="API Gateway Pattern">
MCP configuration is like an API gateway. Each server is a microservice. The agent discovers capabilities and calls what it needs — exactly how enterprise API management works.
</AnalogyBox>
