## Subagents, Hooks, Tool Search
The Claude Agent SDK adds three capabilities: **subagents** (agents spawning agents), **hooks** (lifecycle callbacks), **tool search** (dynamic tool discovery). Key insight: narrow tools beat broad ones — `validate_tax_id` outperforms `process_supplier`.
