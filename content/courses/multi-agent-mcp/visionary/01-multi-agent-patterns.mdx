## Why Multiple Agents?
Single agents hit limits: context overflow, tool overload, inability to hold conflicting perspectives. Multi-agent systems specialize — each agent has a focused role.
<AnalogyBox emoji="🏢" title="Division of Labor">
Like McKesson's MDM team: supplier matching specialist, material master expert, compliance reviewer. Each has their own tools and domain. Multi-agent AI follows the same principle.
</AnalogyBox>
Three patterns: **Pipeline** (sequential, simple), **Hierarchical** (supervisor delegates), **Collaborative** (peers negotiate). Re³ uses hierarchical (Forge supervises) + collaborative (agents debate as peers).
