## MCP: USB-C for AI
MCP standardizes how agents connect to tools with three primitives: **Tools** (functions to call), **Resources** (data to read), **Prompts** (templates to use). One protocol, any tool, any model.
<AnalogyBox emoji="🔌" title="MuleSoft for AI">
MuleSoft connects applications through standardized connectors. MCP connects AI agents to data sources the same way. Register services → discover capabilities → call what you need.
</AnalogyBox>
Apple, OpenAI, and Google all adopted MCP. Understanding it positions you at the infrastructure layer, not just the prompt layer.
