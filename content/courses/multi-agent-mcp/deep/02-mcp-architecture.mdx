## MCP: The Universal Connector for AI

<JargonTip term="MCP">Model Context Protocol</JargonTip> is Anthropic's open standard for connecting AI agents to external tools and data sources. Think of it as **USB-C for AI** — one protocol, any tool, any model.

Before MCP, every AI integration was custom: bespoke API wrappers, custom authentication, fragile prompt-parsing. MCP standardizes this with three primitives.

### MCP's Three Primitives

<ComparisonTable title="MCP Primitives" columns={['Primitive','What It Does','Direction','Enterprise Analogy']} rows={[
  ['Tools','Functions the LLM can call','Server → Client','API endpoints — the actions an agent can take'],
  ['Resources','Data the LLM can read','Server → Client','Database views — structured data available to the agent'],
  ['Prompts','Templates the server suggests','Server → Client','Stored procedures — pre-built workflows the server offers'],
]}/>

<AnalogyBox emoji="🔌" title="MuleSoft Anypoint for AI">
If you've worked with MuleSoft at McKesson, MCP will feel familiar. MuleSoft provides a universal API layer that connects any system (SAP, Salesforce, Oracle) through standardized connectors. ### Building an MCP Server

<CodeBlock language="python" label="Simple MCP Server — Knowledge Base" code={`from mcp.server import Server
from mcp.server.stdio import stdio_server
from mcp.types import Tool, TextContent
import json

app = Server("knowledge-base-server")

# Define a tool the LLM can call
@app.list_tools()
async def list_tools():
    return [
        Tool(
            name="search_knowledge",
            description="Search the internal knowledge base",
            inputSchema={
                "type": "object",
                "properties": {
                    "query": {
                        "type": "string",
                        "description": "Search query"
                    }
                },
                "required": ["query"]
            }
        )
    ]

@app.call_tool()
async def call_tool(name: str, arguments: dict):
    if name == "search_knowledge":
        query = arguments["query"]
        # In production: query your actual knowledge base
        results = search_internal_kb(query)
        return [TextContent(type="text", text=json.dumps(results))]

# Run with stdio transport
async def main():
    async with stdio_server() as (read, write):
        await app.run(read, write, 
            app.create_initialization_options())

if __name__ == "__main__":
    import asyncio
    asyncio.run(main())`}/>

### Why MCP Matters Now

Apple added MCP support to Xcode. OpenAI added it to ChatGPT. Google is integrating it. This isn't a niche Anthropic thing — it's becoming the industry standard for AI tool connectivity. Understanding MCP positions you as someone who grasps the infrastructure layer, not just the prompt layer.
