## Week 3 Build: 3-Agent Collaboration System

Build a Researcher → Analyst → Writer pipeline where each agent has specialized tools via MCP. Connect at least one MCP server (filesystem, web search, or a custom API wrapper).

### Requirements Checklist

1. Three specialized agents: Researcher, Analyst, Writer
2. Supervisor node that decides agent execution order
3. Each agent has its own system prompt and focused role
4. At least one MCP server connected (filesystem is easiest to start)
5. Quality gate after Writer — loop back for revision if score < 7
6. State tracks each agent's output independently
7. Final output is a structured report

### Stretch Goals

1. **Parallel execution**: Run Researcher + secondary Researcher in parallel using LangGraph's `Send` API
2. **Agent memory**: Add short-term memory so agents can reference each other's previous outputs
3. **MCP server**: Build a custom MCP server that wraps one of your existing APIs (Open Food Facts, supplier database)
4. **Human-in-the-loop**: Add `interrupt()` before the final output for human review
