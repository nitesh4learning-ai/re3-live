## Routing Strategies

Different routing strategies optimize for different goals. The gateway also manages <JargonTip term="streaming">streaming</JargonTip> connections to reduce perceived latency. The best strategy depends on your priorities.



| Strategy | How It Works | Best For |
|----------|-------------|----------|
| Cost-Optimized | Route to cheapest available provider | Budget-constrained applications |
| Latency-Optimized | Route to fastest available provider | Real-time applications, chatbots |
| Quality-Optimized | Route to best model for the task type | High-stakes applications |
| Round-Robin | Distribute evenly across providers | Load balancing, rate limit spreading |
| Fallback Chain | Try primary, fall back to secondary | Maximum reliability |
