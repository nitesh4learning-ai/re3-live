## ISO 42001 Structure



<JargonTip term="ISO 42001">ISO/IEC 42001:2023</JargonTip> is the world's first international standard for **AI Management Systems (AIMS)**. It provides a certifiable framework using Plan-Do-Check-Act methodology — the same structure as ISO 27001 (information security) and ISO 9001 (quality management).

<AnalogyBox emoji="📜" title="ISO 27001 for AI">
If your organization has ISO 27001 certification, ISO 42001 will feel familiar. Same PDCA cycle, same management system approach, same audit structure. The difference: 27001 manages information security risks. 42001 manages AI-specific risks like bias, transparency, and autonomous decision-making.
</AnalogyBox>
