## SHAP: Feature Attribution via Game Theory

SHAP calculates each feature's fair contribution to a prediction using Shapley values. Positive values push prediction higher, negative push lower. TreeExplainer is fast for tree models; KernelExplainer works on anything but is slow.

<AnalogyBox emoji="⚖️" title="Fair Division of Credit">
Three sales reps close a deal. How to divide commission fairly? Shapley values calculate each person's marginal contribution across all team combinations. SHAP does the same for model features.
</AnalogyBox>
