## Measuring Agent Quality: LangSmith & Custom Evals

You can't improve what you don't measure. Agent evaluation answers three questions: **Is the output correct?** (accuracy), **Is it worth the cost?** (efficiency), and **Does it degrade over time?** (monitoring).

### LangSmith for Agent Tracing

<CodeBlock language="python" label="Adding LangSmith Tracing">
{`import os
os.environ["LANGCHAIN_TRACING_V2"] = "true"
os.environ["LANGCHAIN_API_KEY"] = "your-langsmith-key"
os.environ["LANGCHAIN_PROJECT"] = "snapsense-agent-eval"

# That's it - LangGraph automatically traces every node,

### Custom Evaluation Framework

<CodeBlock language="python" label="SnapSense Scan Evaluation Suite">
{`# Test cases: (product, expected_category, expect_rating, expect_safety)
TEST_CASES = [
    ("Tylenol Extra Strength 500mg", "Medicine", False, True),
    ("Organic Baby Spinach", "Food", True, False),
    ("TERRO Ant Bait", "Household Chemical", False, True),
]

def run_eval(agent_fn):
    results = []
    for (product, exp_cat, exp_rating, exp_safety) in TEST_CASES:
        output = agent_fn(product)
        cat_correct = output.get("category") == exp_cat
        rating_correct = bool(output.get("rating")) == exp_rating
        results.append((product, cat_correct, rating_correct))

    correct = sum(1 for r in results if r[1])
    total = len(results)
    pct = int(100 * correct / total)
    print("Accuracy: " + str(pct) + "%")
    print("Passed: " + str(correct) + "/" + str(total))
    return results`}
</CodeBlock>

<AnalogyBox emoji="📊" title="Data Quality Scorecard">
This evaluation framework is the AI equivalent of your MDM data quality scorecard — measuring accuracy, completeness, timeliness, and conformity. Instead of golden record quality, you're measuring agent output quality. The principles are identical; the domain is different.
</AnalogyBox>
