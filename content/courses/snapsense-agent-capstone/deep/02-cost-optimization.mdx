## Agent Cost Architecture: Spend Smart, Not More

Production agents can be expensive if every decision routes through a frontier model. The key is **model routing** — use the cheapest model that meets quality requirements for each task.

### The Model Routing Hierarchy

<ComparisonTable title="Model Routing for SnapSense" columns={['Task','Model','Cost/1K tokens','Why This Model']} rows={[
  ['Category classification','Gemini Flash','~$0.003','Fast, cheap, good enough for 16-category routing'],
  ['Smart Capture tiles','Haiku','~$0.005','Structured output, moderate quality needed'],
  ['Deep Analysis','Sonnet','~$0.015','Complex reasoning, nuanced insights required'],
  ['Quality grading','Haiku','~$0.005','Binary good/bad decision, speed matters'],
  ['Query rewriting','Haiku','~$0.005','Simple text transformation'],
]}/>

<AnalogyBox emoji="🏗️" title="Tiered Support Model">
This is like IT support tiers: L1 handles password resets (cheap, fast), L2 handles complex issues (moderate cost), L3 involves senior engineers (expensive, reserved for critical problems). You don't send password resets to L3. You don't route category classification to Opus.
</AnalogyBox>

### Cost Tracking in State

<CodeBlock>
{`from typing import TypedDict, List

class AgentState(TypedDict):
    # ... existing fields ...
    
    # Cost tracking
    total_tokens_input: int
    total_tokens_output: int
    total_cost_usd: float
    model_calls: List[dict]  # [{'{'{'}'}model, tokens, cost, node{'}'}]
    
def track_cost(state, model, input_tokens, output_tokens, node_name):
    """
    Track per-call cost for observability.
    Critical for proving ROI to leadership.
    """
    # Approximate costs per 1K tokens (input/output)
    COSTS = {'{'{'}'}
        "gemini-flash": (0.003, 0.006),
        "claude-haiku": (0.005, 0.01),
        "claude-sonnet": (0.015, 0.06),
    {'}'}
    in_rate, out_rate = COSTS.get(model, (0.01, 0.03))
    cost = (input_tokens * in_rate + output_tokens * out_rate) / 1000
    
    return {'{'{'}'}
        "total_tokens_input": state["total_tokens_input"] + input_tokens,
        "total_tokens_output": state["total_tokens_output"] + output_tokens,
        "total_cost_usd": state["total_cost_usd"] + cost,
        "model_calls": state["model_calls"] + [{'{'{'}'}
            "model": model, "node": node_name,
            "tokens": input_tokens + output_tokens,
            "cost": round(cost, 6)
        {'}'}]
    {'}'}`}
</CodeBlock>
