## Week 4 Capstone: SnapSense Agent Layer

Design and prototype an agent layer for SnapSense. Even if you don't ship it immediately, the architecture documentation is your interview portfolio piece.

### Deliverables

1. **Architecture Decision Record** — ADR documenting the agent layer design
2. **Working Prototype** — LangGraph graph with at least Category Agent + Quality Gate
3. **Cost Model** — Per-scan cost breakdown with model routing
4. **Evaluation Suite** — 10+ test cases covering known edge cases (TERRO, Tylenol, sabudana)
5. **LangSmith Dashboard** — Tracing setup showing agent execution flow

### Stretch Goals

1. Deploy the prototype as a FastAPI endpoint
2. Add cost alerts when per-scan cost exceeds threshold
3. Build a comparison dashboard: current pipeline vs agent pipeline (accuracy, cost, latency)
4. Create a presentation deck summarizing Phase 1 learnings for leadership
