## From Theory to Product: The SnapSense Agent Layer

Weeks 1-3 taught you LangGraph, Agentic RAG, and Multi-Agent patterns. This week, you apply everything to a real product: SnapSense AI.

### The Current SnapSense Pipeline

<CodeBlock>
{`Camera Scan вҶ’ ML Kit (on-device) вҶ’ Cloud Vision вҶ’ LLM (Gemini/Haiku)
                                                        вҶ“
                                               Smart Capture Tiles
                                                        вҶ“
                                               Deep Analysis`}
</CodeBlock>

This is a linear pipeline вҖ” no self-correction, no quality evaluation, no dynamic routing. If ML Kit misclassifies an object (sabudana as "plant"), the entire downstream output is wrong.

<AnalogyBox emoji="рҹҸӯ" title="MDM Data Pipeline vs AI Agent Pipeline">
The SnapSense agent layer mirrors an MDM data pipeline: ingest (camera scan) вҶ’ classify (category routing) вҶ’ validate (data quality grading) вҶ’ enrich (API lookups) вҶ’ quality gate (human review for edge cases) вҶ’ publish (generate tiles). Every MDM pipeline principle applies here.
