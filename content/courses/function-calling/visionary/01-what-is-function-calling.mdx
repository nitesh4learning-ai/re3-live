## What is Function Calling?

<JargonTip term="function calling">Function calling</JargonTip> lets an LLM **decide when to use external <JargonTip term="tool use">tools</JargonTip>** and generate the structured arguments needed to call them. The LLM doesn't execute the function directly -- it outputs a JSON request that your code then executes.

<AnalogyBox emoji="🍽️" title="Think of it like a restaurant waiter">The waiter (LLM) takes your order and decides which kitchen station (function) should prepare it. The waiter doesn't cook -- they route the request to the right place with the right instructions.</AnalogyBox>
