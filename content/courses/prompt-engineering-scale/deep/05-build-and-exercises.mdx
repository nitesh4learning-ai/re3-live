### Week 11 Build: SnapSense Prompt Evaluation Suite

**Deliverables:**
1. Formalized prompt templates for all 16 SnapSense categories
2. Evaluation suite with 20+ test cases per category
3. A/B test harness: compare prompt variations on accuracy/cost/latency
4. Model routing matrix: which model tier for which task
5. Cost optimization report: current spend vs optimized spend

> **Interview soundbite**: *"Most people 'do prompting.' I built a systematic evaluation framework — 20+ test cases per category, A/B testing across model tiers, cost optimization that reduced per-scan spend by 30%. That's engineering discipline, not enthusiasm."*
