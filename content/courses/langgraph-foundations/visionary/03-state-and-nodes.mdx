## State: Your Agent's Memory

State is a Python `TypedDict` — the shared whiteboard every node reads and writes. Design it like an MDM master record: core attributes, decision fields, and audit metadata.

<AnalogyBox emoji="📋" title="Master Data Record">
A master record has core attributes (name, address), derived fields (match score), and audit fields (source_system). LangGraph state follows the same pattern: core data, decision fields, and observability metadata.
</AnalogyBox>

### Nodes: Single-Responsibility Workers

Each node does **one thing**: search, grade, rewrite, or summarize. Nodes return **partial state updates** (dicts) — LangGraph merges them with existing state. You patch, not replace.
