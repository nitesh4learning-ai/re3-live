## The SAP AI Toolkit

SAP ABAP-1 (foundation model, 250M lines of ABAP), Joule for Developers (IDE-integrated, free until Sep 2026), ABAP MCP Server (GA Q2 2026, connects external agents to SAP), and Claude Code for external architecture analysis. Use Joule for ABAP-specific understanding, Claude Code for strategic modernization planning.
