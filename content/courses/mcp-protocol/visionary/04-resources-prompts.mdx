## Resources & Prompts

MCP has three primitives. Beyond Tools, there are **Resources** (read-only data) and **Prompts** (reusable templates).

| Primitive | Direction | Purpose | Example |
|-----------|-----------|---------|---------|
| **Tools** | AI → Server | Execute actions | Search DB, send email, create file |
| **Resources** | Server → AI | Provide read-only data | Config files, documentation, schemas |
| **Prompts** | Server → User | Reusable templates | Code review template, analysis prompt |
