## Provider Implementation

Each LLM provider implements <JargonTip term="structured output">structured outputs</JargonTip> differently. Some use <JargonTip term="function calling">function calling</JargonTip> under the hood, while others support native <JargonTip term="constrained decoding">constrained decoding</JargonTip>.

client = OpenAI()

response = client.chat.completions.create(

    response_format={
        "type": "json_schema",
        "json_schema": {
            "name": "sentiment_analysis",
            "strict": True,  # Guarantees exact schema match

                "required": ["sentiment", "confidence", "key_phrases"],

    },
