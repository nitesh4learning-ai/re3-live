## Cross-Modal Reasoning
  <ExpandableSection title="What is Cross-Modal Reasoning?" icon="🧠" defaultOpen>

Cross-modal reasoning is when an AI uses information from one modality to inform understanding in another. Models create shared <JargonTip term="embedding">embedding</JargonTip> spaces where different modalities can be compared. For example: reading a chart image (vision) to answer a text question, or using audio context to disambiguate text.
